# Screenshots

Drop PNG (or JPG) files into this folder using the exact filenames below — the root README references them.

| File | What it shows |
|---|---|
| `home-feed.png` | Dashboard home feed |
| `assignments.png` | Assignments list page |
| `generate-assignment.png` | Create / generate assignment form |
| `paper-generated.png` | Generated paper view |
| `toolkit.png` | AI Teacher's Toolkit page |

Tips:
- Capture at standard desktop width (~1440px) for consistent sizing.
- Trim browser chrome (or use a clean theme) before saving.
- Keep files under ~500 KB each — compress with [Squoosh](https://squoosh.app/) if needed.
